## 🎮 关于项目

**Resonance Geometry** 是一款创新的3D节奏建筑游戏，将音乐节奏与几何建筑完美融合。玩家通过击打音符来实时构建壮观的几何结构，每一首歌曲都是一次独特的建筑创作体验。

## ✨ 核心特色

- **🎵 音乐驱动构建** - 音符沿着3D轨道流动，击打它们来构建几何建筑
- **🏗️ 实时建筑生成** - 观看你的建筑随着音乐节奏同步生长
- **🎯 动态镜头系统** - 智能镜头环绕展示建筑的每个角度
- **🌟 视觉反馈系统** - 连击和精准度影响建筑的华丽程度和光芒强度
- **🎨 霓虹美学风格** - 低多边形几何体与动态流光效果的完美结合

## 🚀 快速开始

### 系统要求
- **Unity 2022.3** 或更高版本
- **Windows 10/11** 或 **macOS 12+**

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/gemjun/Resonance-Geometry.git
   cd Resonance-Geometry
   ```

2. **使用Unity Hub打开项目**
   - 选择克隆的文件夹作为项目路径

3. **导入必要资源**
   - 项目首次打开时会自动导入所需资源包

4. **开始体验**
   - 打开 `Assets/Scenes/MainScene.unity`
   - 点击运行按钮开始游戏！

## 🎯 游戏玩法

1. **节奏击打** - 当音符沿着3D轨道接近时，在正确时机按下对应键位
2. **建筑构建** - 每个成功击打的音符都会转化为建筑的一部分
3. **连击奖励** - 保持连击来获得分数乘数和更华丽的视觉效果
4. **多视角体验** - 游戏会自动切换视角，从不同角度展示你的建筑杰作
